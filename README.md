Ever found yourself wanting a refresher for some information but scared that going to the wiki would expose you to potential spoilers?

JJK Companion Handbook is a companion app that helps you keep track of info as you watch episode-by-episode!
It summarizes need-to-know information about details that have been explicitly revealed by the show, based on the 'Currently Watching' episode, up to the previous episode.
Information is faithful to what the show reveals, no need to worry about spoilers (unless you skip episodes).
